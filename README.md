#Что Может?------------------------------------------------------------------------------------------------------------------------------------------------

На данный момент FFM может:
-Удалять файлы-каталоги
-Создавать каталоги
-Открывать файлы любым доступным на компьютере способом
-Перемещать файлы и директории по всей файловой системе
-Копировать файлы и директории
-Переименовывать файлы и директории
-Разархивировать архивы tar и zip
-Ну и естественно дает обзор над всей файловой системой
